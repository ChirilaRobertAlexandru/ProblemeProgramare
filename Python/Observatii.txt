CITIRE DATE
	->In Python, poți citi un număr de la tastatură folosind funcția input().
		Această funcție va citi totul ca un șir de caractere (string), astfel că este necesar să convertești acest șir la tipul de date numeric dorit, cum ar fi int sau float.
		EXEMPLU:
				numar = int(input("Introduceți un număr: ")) //citeste un numar de tip int
	->Citire multipla:
		a, b = map(int, input().split())
		// map face conversia implicita int fiecarui element din lista
		// .split() imparte șirul citit într-o listă de cuvinte folosind separatorul implicit (spațiu).

	-> Operatori de identitate:

				2. is not
				Exemplu:
						x = ["apple", "banana"]

						print(x is z)	#True , pentru ca z este acelasi obiect ca si x
